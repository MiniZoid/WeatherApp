# WeatherApp

A fork of https://github.com/MiniZoid/WeatherApp that adds Gradle support.

## Run Tests:

### Mac/Linux:

```shell
$ ./gradlew test
```

### Windows:

```shell
$ gradlew.bat test
```

## Run App

### Mac/Linux:

```shell
$ ./gradlew run
```

### Windows:

```shell
$ gradlew.bat run
```

## Build App

### Mac/Linux:

```shell
$ ./gradlew build
```

### Windows:

```shell
$ gradlew.bat build
```

## Running Built App

### Mac/Linux/Windows:

```shell
$ java -jar build/libs/WeatherApp-0.1.jar
```
